\textbf{Порождение и запуск процессов}

Выполнить задание для исключения: ошибка деления на ноль и для заданного выбранного исключения -- выполнение несуществующей инструкции. На каждый пункт представить отдельную программу, специфический код, связанный с особенностями генерации заданного исключения структурировать в отдельный элемент (функцию, макрос или иное).

\begin{enumerate}
	\item Сгенерировать и обработать исключения с помощью функций WinAPI. Получить код исключения с помощью функции \code{GetExceptionCode}.
	\begin{itemize}
		\item Использовать эту функции в выражении фильтре;
		\item Использовать эту функцию в обработчике.
	\end{itemize}
	\item Создать собственную функцию-фильтр.
	\item Получить информацию об исключении с помощью функции \\ \code{GetExceptionInformation}. Сгенерировать исключение с помощью функции \code{RaiseException}.
	\item Использовать функции \code{UnhandleExceptionFilter} и \\ \code{SetUnhandleExceptionFilter} для необработанных исключений.
	\item Обработать вложенные исключения.
	\item Выйти из блока \code{__try} с помощью операторов \code{goto} и \code{__leave}.
	\item Преобразовать структурное исключение в исключение языка \code{C++}, используя функцию \code{translator}.
	\item Использовать финальный обработчик \code{__finally}.
	\item Проверить корректность выхода из блока \code{__try} с помощью функции \code{AbnormalTermination} в финальном обработчике \code{__finally}.
\end{enumerate}
